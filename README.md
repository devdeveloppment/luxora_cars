# Luxora Cars - Marketplace de Luxe 🚗✨

Luxora Cars est une plateforme premium dédiée aux supercars et véhicules d'exception. 

## Fonctionnalités
- ✨ Catalogue de véhicules de luxe (Rolls-Royce, Ferrari, McLaren, Lamborghini...)
- 📊 Recherche multicritère (Marque, Budget, Type)
- 💳 Système de paiement sécurisé via CinetPay
- 🛠️ Interface d'administration pour la gestion des stocks
- 📱 Design moderne, premium et entièrement responsive

## Technologies
- **Backend**: Django (Python)
- **Frontend**: HTML5, CSS3, JavaScript (Vanilla)
- **Base de données**: SQLite / PostgreSQL
- **Paiements**: Intégration CinetPay

## Installation
1. Clonez le projet : `git clone https://github.com/devdeveloppment/luxora_cars.git`
2. Installez les dépendances : `pip install -r requirements.txt`
3. Lancez les migrations : `python manage.py migrate`
4. Exécutez le serveur : `python manage.py runserver`

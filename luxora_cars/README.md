# LUXORA CARS — Application Web Premium

## Description
Site web de vente de voitures de luxe avec intégration CinetPay.

## Contenu du projet
- `index.html` — Application complète (tout-en-un)

## Fonctionnalités
- Page d'accueil avec fond luxueux (Lamborghini Aventador)
- Catalogue de 16 véhicules de luxe avec vraies photos par marque
- Filtres avancés (marque, prix, carburant, carrosserie, année)
- Fiche véhicule avec galerie photos (5 images par véhicule)
- Système de paiement CinetPay (Wave, Orange Money, MTN MoMo, Carte)
- Formulaire de vente en 4 étapes
- Dashboard admin complet
- Espace utilisateur (favoris, commandes, annonces)
- Chat en direct

## Lancement local
Ouvrez simplement `index.html` dans votre navigateur.
Pour le paiement CinetPay, un serveur HTTP est recommandé :
```
python -m http.server 8080
```
Puis ouvrez http://localhost:8080
